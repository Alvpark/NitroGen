## Features

- Generating Nitro Codes
- Online Check
- Online Check With Proxy (Only http proxies)

## Installation

**Required:** Node.js [axios, chalk, clear, figlet, fs, inquirer]
Clone Repository

```bash
  git clone https://github.com/mrawsky/NitroGen.git
  cd NitroGen
```

Install Node Packages

```bash
  npm i
```

Run NitroGen

```bash
  node ./index.js
```
